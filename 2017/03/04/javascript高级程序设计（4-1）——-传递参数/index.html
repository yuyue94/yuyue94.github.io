javascript高级程序设计（4.1）—— 传递参数

标签（空格分隔）：javascript

拾遗

基本类型和引用类型的值

  • 基本类型有5种：Undefined、Null、Boolean、Number、String
  • 引用类型：Object

传递参数

  • 参数的传递在ECMAScript中永远是按照值传递，不存在按照引用传递
  • 在传递对象时，由于局部变量复制的是传递过来的对象的引用（但这个引用依然是值传递，下面代码来验证），所以在函数内对对象的新属性赋值，对外部依然有效。

检查类型

  • 检测基本类型用typeof即可搞定，但是检测引用类型用处不大
  • typeof一个函数（类）返回”function”，typeof一个对象返回”object”
  • RegExp是个类，typeof返回”function”，/^aaa/是个正则对象，typeof返回”object”（在某些浏览器的早起版本，由于正则对象实现了[[call]]方法，使得typeof正则对象返回”function”，现在可以不予考虑）
  • 检测引用类型使用instanceof操作符，如果变量是给定引用类型的实例，那么instanceof会返回true
