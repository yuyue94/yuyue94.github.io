语句

if语句

  • if(condition)其实会自动调用Boolean()函数来转化判断真值

  • 情况1：关系操作符==，当不为Number型时，自动调用Number()函数转换，Numer([])为0，Number(false)为0，关系值为true

  • 情况2：无关系操作符，直接Boolean([])为true

for-in语句

with语句（不建议使用）

函数

理解参数

  • 不管函数定义中是否有参数设置，你调用函数传入的值都会被存在arguments这个array-like object中，它是object不是list，所以要使用array的各种方法，可以使用如下的语句

  • arguments里面的值永远与对应命名参数的值保持一致。前提是该值确实传入（arguments的length由传入参数的个数决定），两者内存空间却是独立的，但值会同步

  • 严格模式下arguments的值不能被改变

没有重载

  • 函数名唯一标志了一个函数，不像java中可以根据不同参数重载函数，函数名属于后定义的函数（如果函数名相同）
